# 💰 Smart Finance Analyzer Pro

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://tu-app-url.streamlit.app/)
[![Python 3.9+](https://img.shields.io/badge/python-3.9+-blue.svg)](https://www.python.org/downloads/)

**Smart Finance Analyzer Pro** es una solución integral de análisis de finanzas personales que transforma extractos bancarios brutos (CSV) en dashboards interactivos con inteligencia predictiva.



## 🌟 Características Principales

* **📊 Análisis Descriptivo:** Clasificación automática de gastos mediante un motor de lógica semántica.
* **🔮 Predicciones con IA:** Implementación de un modelo de **Regresión Lineal (Scikit-learn)** para estimar el flujo de caja del próximo mes.
* **🕵️ Detector de Suscripciones:** Algoritmo de detección de patrones recurrentes para identificar servicios de pago mensual.
* **🚨 Detección de Anomalías:** Análisis estadístico para alertar sobre gastos fuera del patrón habitual del usuario.
* **📥 Exportación Pro:** Generación de reportes limpios y descarga de datos procesados.

## 🛠️ Stack Tecnológico

* **Lenguaje:** Python
* **Manipulación de Datos:** Pandas & NumPy
* **Machine Learning:** Scikit-learn (Linear Regression)
* **Visualización:** Plotly Express
* **Interfaz:** Streamlit
